# AGENTS.md File

## Memory

Use `.agent-memory` directory as memory for storing and remembering things between conversations.

## Context

For additional context, refer to the `.agent-context` directory.

## GitHub Issues

When working on an issue, if you don't have information about what needs to be done, refer to the GitHub issue list. The issue tracking information is stored in `.agent-memory/issues.json`, which contains details about each issue including:
- Issue number and title
- Status (open/closed)
- Phase assignment
- GitHub URL for the full issue description
- PR links (if applicable)
- Notes and completion dates

Always check the GitHub issue URL for the complete requirements and acceptance criteria before starting work on an issue.
